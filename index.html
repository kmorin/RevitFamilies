Revit Families

Up-To-Date Revit Families via Git

Download .zip Download .tar.gz View on GitHub

Here is a maintained, up-to-date repository of useful (and not so useful) Revit families for your projects.

If you would like to add to this growing list as a contributor, please fork and/or join this repo as a contributor.

How to get started

Easiest

  1. Click on the zipball link above^ to get a current set of rfa families.

Better

  1. Fork the repository in your GitHub account.
  2. Download and install GitHub for Windows (http://windows.github.com/)
  3. Using the GitHub for Windows client, clone the repo to your computer
  4. Add families of your own to the master branch!
